# 01 Bridger (Nerves) Organ
**Role**: CONNECT
**Biological**: Nerves (Ganglia)
**Greek**: Logos (Word/Reason/Connection)

This organ is responsible for **Connecting** the system. It manages:
- The NATS Bus (Message Passing)
- MCP Interfaces (Model Context Protocol)
- Translation layers between components.

## Contents
- `bridger.py`: The core bridging logic.
- `adapters/`: Adapters for NATS and File System.

# 02 Shaper (Hands) Organ
**Role**: ACT
**Biological**: Hands (Mandibles)
**Greek**: Techne (Art/Craft/Skill)

This organ is responsible for **Execution**. It contains:
- MCP Servers (Tools exposed to the LLM)
- Workflows (Pre-defined sequences of actions)
- CLI Tools and Scripts.

## Contents
- `servers/`: MCP Server implementations (e.g., `search_mcp.py`).
- `workflows/`: Complex multi-step workflows (e.g., `research_workflow.py`).

# 06 Assimilator (Memory) Organ
**Role**: MEMORY
**Biological**: Memory (Mycelium)
**Greek**: Topos (Place/Location)

This organ is responsible for **Memory and Storage**. It houses:
- LanceDB (Vector Memory)
- SQLite (Relational Memory)
- The Knowledge Graph
- Ingestion Scripts.

## Contents
- `assimilator.py`: Core logic for memory assimilation.
- `ingest_*.py`: Scripts to ingest data into memory.
- `query_memory.py`: Tools to query the memory.
- `lancedb/`: The Vector Database files.
- `library_diataxis/`: The Crystallized Knowledge Library.
